## Getting Help

<div className="fd-steps [&_h3]:fd-step">

### üîç Check Documentation First
Our documentation covers most common questions and issues. Start by searching the docs for relevant information.

- Use the search function at the top of the documentation
- Check the FAQ section for common questions
- Review the getting started guide for basics

### üí¨ Ask the Community
If you can't find an answer in the documentation, reach out to the community.

- Post in the appropriate channel on Discord
- Create a new topic on the forum
- Check if someone has already asked a similar question

### üêõ Report Issues
If you believe you've found a bug or have a specific technical problem:

- Check GitHub issues to see if it's already reported
- Create a new issue with steps to reproduce
- Include system information and error messages

### üìß Contact Support
For sensitive matters or issues that require staff attention:

- Email support@bestoflinux.org
- Include relevant details about your issue
- Allow 1-2 business days for a response

</div>
